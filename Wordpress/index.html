🐾 PetCare – WordPress Website

A fully responsive and user-friendly pet care site crafted with free tools and the Astra theme.

📘 Project Overview

PetCare is a clean, fully responsive WordPress website made for a fictional pet care service. Built using the free Astra theme and plugins, it demonstrates my ability to create professional layouts using only open-source tools.

🧩 Key Features

  • Mobile-friendly responsive design
  • Customized layout using Astra Theme
  • Functional contact form with Ninja Forms
  • Spam protection via Antispam Bee
  • Multi-page structure: Home, About, Services, Contact

Frontend Developer & WordPress Customizer
